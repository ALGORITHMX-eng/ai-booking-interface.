# AI Systems Architecture — Booking Interface
A high-fidelity, production-ready scheduling interface designed for AI architects.

### 🏗️ Technical Architecture
- **Dynamic Date Logic:** Vanilla JS engine that calculates month offsets and leap years without external dependencies (Moment.js/Day.js).
- **Glassmorphism UI:** Advanced CSS `backdrop-filter` implementation with a custom Obsidian-base color palette.
- **Responsive Systems:** 12-column fluid grid system optimized for both mobile-first consultation booking and desktop views.
- **State Management:** Real-time UI state updates for time-slot selection and CTA visibility.

### 🛠️ Tech Stack
- **Language:** HTML5, CSS3 (Variables/Grid/Flex), JavaScript (ES6+)
- **Theme:** Obsidian × Neon Cyber-SaaS
